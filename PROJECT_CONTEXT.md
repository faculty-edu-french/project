# سياق المشروع (Project Context)
**هذا الملف مخصص لي كـ AI (Antigravity/Claude/Gemini) عند بدء محادثة جديدة، لكي أفهم حالة المشروع ومساره وما تم إنجازه، وأستأنف العمل فوراً.**

## 1. ما هو هذا المشروع؟
*   **الاسم/الهدف:** منصة تعليمية تفاعلية (Web-Book/Livret Numérique) لطلاب كلية التربية - قسم اللغة الفرنسية (جامعة المنيا).
*   **التقنيات المستخدمة:** React, TypeScript, Vite, CSS (بدون إطار CSS مثل Tailwind، نعتمد على Vanilla CSS بتصميم فخم ومستجيب "فشيخ").
*   **هيكلة المحتوى:** المحتوى التعليمي لا يُكتب مباشرة في الـ Components، بل يُقرأ من ملفات JSON (مثل `module1_refined.json` و `module2_refined.json`). الـ `App.tsx` يقوم بعمل Loop على هذه الـ blocks ويرسمها بناءً على الـ `type` الخاص بكل block (مثل: `task`, `instruction`, `grammar_card`, `input`, `video`، إلخ).

## 2. ماذا أنجزنا حتى الآن؟
*   **هيكلة الدروس:** تم نقل وضبط دروس الوحدة الأولى (Module 1) والوحدة الثانية (Module 2) داخل ملفات الـ JSON.
*   **حفظ الإجابات المتقدم:** أضفنا ميزة حفظ إجابات الطالب (text inputs) محلياً في المتصفح باستخدام `localStorage` لكي لا تُفقد الإجابات عند إعادة تحميل الصفحة.
*   **إرسال الإجابات للأستاذ:** الطريقة المعتمدة حالياً هي إرسال نموذج إجابات الطالب (EssaySubmitter) إلى **Telegram Bot** خاص. تم حل مشكلة الرسائل الطويلة (أكثر من 4096 حرف) بتقسيمها برمجياً لضمان عدم حدوث عبارة "Erreur de connexion".
*   **التصميم (UI/UX):** 
    *   أضفنا تصميم بطاقات (Cards) احترافي زجاجي (Glassmorphism) لشرح القواعد Grammar بطريقة بصرية ممتازة.
    *   تم تغيير روابط المصادر الخارجية المعطلة، وإنشاء صفحة HTML محلية (`testomdex.html`) تُفتح كنافذة خارجية لقراءة النصوص.
*   **ميزة الطباعة (PDF):** أضفنا زر `🖨️ Imprimer / PDF` في نهاية كل درس. تم ضبط ملف الـ CSS خصيصاً عبر `@media print` لإخفاء القوائم والحواف الجانبية وإخراج الدرس في صفحات PDF نظيفة ومرتبة بالكامل.

## 3. كيف نقوم برفع التعديلات (Deployment)؟
الملف الحي للمشروع مرفوع على **GitHub Pages**. أي تعديل يتم في الكود أو ملفات الـ JSON لن يظهر للمستخدمين إلا بعد عمل Deploy.
الخطوات التي أقوم بها كـ AI لرفع أي تعديل:

1.  أدخل لمسار المشروع: `cd /home/george/Desktop/project/web-book`
2.  أُشغل أوامر البناء والنشر معاً: 
    `npm run predeploy && npm run deploy`
    *(ملاحظة: هذا الكوماند يترجم التايب سكريبت ويبني مجلد dist ويدفعه لفرع gh-pages).*
3.  ثم أقوم بحفظ نفس التعديلات على الـ Repository لحفظ المصدر:
    `git add . && git commit -m "update message" && git push origin main`
